React + TypeScript Portfolio Blog

A responsive portfolio blog application built with React, TypeScript, and Vite.  
The project demonstrates modern frontend development practices including component-based architecture, API data fetching, responsive design, and live deployment.

 Live Demo
 https://YOUR-VERCEL-LINK.vercel.app

Features
- Blog posts fetched from an external API
- Reusable React components
- Type-safe code with TypeScript
- Responsive layout (desktop & mobile)
- Dark / Light mode support
- Clean and minimal UI
- Deployed live using Vercel

Tech Stack
- React
- TypeScript
- Vite
- CSS
- JSONPlaceholder API
- Git & GitHub
- Vercel (Deployment)
